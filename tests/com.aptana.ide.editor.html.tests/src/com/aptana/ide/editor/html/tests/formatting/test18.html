Preserve Ratio
For some things like images, you will probably want to preserve the ratio of width to height. Just set preserveRatio:true. 

var wrapped = new Ext.Resizable('wrapped', {
    wrap:true,
    pinned:true,
    minWidth:50,
    minHeight: 50,
    preserveRatio: true
});
