Changing the Aptana Studio interface to a different language [edit]

This page discusses how to switch the Aptana Studio interface to a different langauge, such as Japanese or Chinese

Introduction

You can change the language of the Aptana UI to a different language than what your operating system uses by default. Follow the instructions below to change the language of the Aptana UI.

Instructions

Windows Users

You can do either of the following to change the language of your Aptana UI:

  • Change your language of your operating system, or
  • Change your shortcut to Aptana by typing the following on the command line when you start Aptana (choose your two-letter language abbreviation from the list at the bottom of this page): 
Aptana.exe -nl [2-letter-language-abbreviation]

Mac OS users

To change the language of your UI:

  1. Quit Aptana, if it is running.
  2. Navigate to the directory where you have your Aptana.app installed.
  3. Right-click or Ctrl-click Aptana.app, and select Show Package Contents > Content > MacOS from the context menu.
  4. Open the Aptana.ini file in a text editor.
  5. At the first line of the Aptana.ini file, type the following:

    -nl
[2-letter-abbreviation]

    Make sure that you have a new line between the "-nl" and the abbreviation. Choose your abbreviation from the list at the bottom of this page.

  6. Save your changes and close the file.
  7. Re-start Aptana. Your UI should be in the language that you chose.

Two-letter language abbreviations

Aptana recognizes the following language abbreviations:

  • en (English)
  • es (Spanish)
  • de (German)
  • fr (French)
  • hu (Hungarian)
  • ja (Japanese)
  • ko (Korean)
  • pl (Polish)
  • ru (Russian)
  • zh (Chinese)

The -nl gives you freedom to choose a language manually.

If there is a string which is not translated, you can translate it here: http://translate.aptana.com/
