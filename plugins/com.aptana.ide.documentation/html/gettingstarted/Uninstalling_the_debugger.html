Uninstalling the debugger [edit]

This page describes how to uninstall the Aptana JavaScript debugger from your web browser.

Introduction

If you need to uninstall the Aptana debugger, you can do so directly from Firefox.

Note: To update the debugger extension, please choose the menu option Tools > Add-ons/Extensions, inside Firefox, and choose "Update".

Instructions

To uninstall the debugger from Firefox:

  1. From the Tools menu in Firefox, select Extensions to open an Extensions pop-up window.
  2. On the Extensions pop-up window, select the Debugger extension.
  3. Click the Uninstall button.
  4. Close and re-start Firefox.

The debugger will be uninstalled the next time that you start Firefox.
