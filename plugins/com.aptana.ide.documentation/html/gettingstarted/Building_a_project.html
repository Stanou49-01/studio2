Building a project [edit]

This page gives an overview of how to build a project in Eclipse and Aptana.

Introduction

In Eclipse and Eclipse-based IDEs, such as Aptana, building a project and running a project are often the same thing.

To build and run a project:

  1. Select the project in the Project View or Navigator View.
  2. Click the Run button

If this is the first time that you have built or run that project, see the individual Help topics for configuration help (e.g. Setting up your web browser configuration).
