Integrating RadRails with Aptana [edit]

This page describes how to integrate RadRails into your installed version of Aptana.

Introduction

Add RadRails as a plugin to Eclipse with Aptana to create a well-integrated IDE for JavaScript and Ruby.

Instructions

Please note that SUN JVM 1.5 or later is required. This also includes the necessary RDT version, so please do not install/update RDT separately.

  1. First, install Aptana: [[1]]
  2. Once Aptana is installed and Aptana/Eclipse is restarted, open the Aptana Start Page by going to Help > Aptana Start page...
  3. In the Plugins column, click the Install link under "Ruby on Rails". An Updates dialog will appear.
  4. Choose the Rails plugin(s) that you will be installing.
    1. In the Updates dialog, check the boxes next to the Rails features you'd like to install.
    2. If you receive an error like "Ruby Mylyn Connector feature (Optional) requires ...", expand the tree and unselect the Aptana RadRails Integration checkbox. This is an optional extension for Mylyn users.
  5. Install the plugin
    1. Click the Next > button to go to the Feature License screen.
    2. On the Feature License screen, choose the option for I accept the terms of the license agreement.
    3. Click the Next button to go to the Installation screen.
    4. Click the Finish button.

      After a few seconds, a Verification pop-up window opens.

    5. On the Verification pop-up window, click the Install All button.

      Aptana installs the plugin and prompts you to restart your workbench.

  6. On the prompt to restart your workbench prompt, click the Yes button to restart your workbench. If you click the No button, your changes will not take effect until you restart.

Aptana completes the update installation process. After installing the updates, Aptana will automatically close and re-start. You should now be able to use RadRails as a plugin for Aptana.

Installing RadRails manually

To manually install the RadRails plug-in into Aptana or Eclipse:

  1. Download the latest release from http://update.aptana.com/install/rails/3.2/ and follow the instructions on that page for installing manually.
