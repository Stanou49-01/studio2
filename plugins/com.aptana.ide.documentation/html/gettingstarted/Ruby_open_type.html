Ruby open type [edit]

This page gives an overview of the Ruby open type feature.

Introduction

Use the Ruby Open Type dialogue (shown below) to open a specified type.

To open the Open Type dialogue:

  • Press the Ctrl+Shift+T key shortcut (Windows)
  • Select a class or module in the New Class Wizard.
