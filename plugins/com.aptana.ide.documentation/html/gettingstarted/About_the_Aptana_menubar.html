About the Aptana menubar [edit]

This page describes the features of the Aptana menubar.

The Aptana menubar contains menus to help you perform various tasks related to web development. The list below describes the basic functionality of each menu in the menubar.

  • File - Use the File menu to create new, open, close, save, import, and export files and projects.
  • Edit - Use the Edit menu to undo, redo, copy, paste, and perform search and replace in your current file.
  • Navigate - Use the Navigate menu to go to specific locations in your file or project.
  • Search - Use the Search menu to perform a search or search-and-replace in your code or project.
  • Project - Use the Project menu to view or set project properties.
  • Run - Use the Run menu to preview your project in a browser or debug your code.
  • Scripts - Use the Scripts menu to customize your IDE with JavaScript, HTML, and CSS.
  • Window - Use the Window menu to customize your Views or Perspective.
  • Help - Use the Help menu to access Aptana and Eclipse help.
