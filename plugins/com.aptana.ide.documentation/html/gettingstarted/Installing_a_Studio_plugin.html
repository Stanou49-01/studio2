Installing a Studio plugin [edit]

You can add a lot of functionality to Aptana Studio by installing one of the many compatible plugins--Aptana Studio itself is available as both a standalone application and an Eclipse plugin.

Aptana Plugin Installation

Aptana Studio's most popular and featured partner plugins available for easy installation from the Plugins tab of the My Aptana page, including:

  • Aptana PHP
  • Aptana RadRails
  • Adobe AIR Development Plugin for Aptana Studio
  • Apple iPhone Development Plugin for Aptana Studio
  • Apple iPhone Development Plugin for Aptana Studio
  • Nokia WRT Plugin for Aptana Studio (from Nokia)
  • Many popular Ajax libraries and several utilities

To install any of these simply go to the Plugin tab on My Aptana and click the link labeled Get It underneath the name of the plugin. If the plugin is already installed you'll see a message telling you the plugin is up to date or that a new version is available for download.

Using the Update Manager

For non-Aptana plugins you can use the built-in Eclipse Update Manager:

  1. From the Help menu, select Software Updates -> Find and Install... to open an Install/Update pop-up window.
  2. On the Install/Update pop-up window, choose the Search for new features to install option, and click the Next button.
  3. Set up a new remote site to scan for updates.
    1. Click the New Remote Site... button to open a New Update Site pop-up window.
    2. On the New Update Site pop-up window, type the name of the new plug-in in the site Name text box.
    3. In the URL text box, type the URL for the update site. This URL should be listed on the web site or home page for the plug-in.
    4. Click OK.
    5. Click the Finish button to open an Updates window.
  4. On the Updates window, check the box next to the name of the plug-in, and click the Next button.
  5. Choose the option to accept the terms of the license agreement, and click the Next button.
  6. Click the Finish button.
  7. Click the Install All button.
