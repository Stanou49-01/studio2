Uninstalling RadRails [edit]

This page describes how to uninstall the RadRails plug-in.

Introduction

You can uninstall the RadRails plug-in the same way that you would uninstall any other plug-in to Eclipse or Aptana.

Instructions

To uninstall the RadRails plug-in:

  1. Follow the instructions in Uninstalling a plug-in.
  2. For the steps where you select a plug-in, select the RadRails plug-in.
