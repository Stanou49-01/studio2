Removing or Disabling the PHP Plugin [edit]

Most Recent or Only Version

The latest version can only be disabled, not uninstalled.

  1. Help -> Software Updates -> Manage Configuration,
  2. Open the Studio tree
  3. Select the Aptana PHP Development Environment
  4. Click Disable

Older Versions

Older Versions can be uninstalled

  1. Help -> Software Updates -> Manage Configuration
  2. Click the ‘Show Disabled’ toolbar button
  3. Open the Studio tree
  4. Select an older version of the Aptana PHP Development Environment (it will have a red circle with slash on the plugin icon)
  5. Click Uninstall
