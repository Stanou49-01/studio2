Creating a new Studio project that includes an AJAX library [edit]

This walkthrough shows you how to create a new web project that includes an Ajax library, such as Yahoo UI, jQuery, Dojo, EXT JS or Prototype.

  1. From the File menu, choose New Project
  2. On the New Project wizard, select Default Web Project in the Aptana Projects folder and click Next
  3. In the Project name field, enter a name for your new project and click Next
  4. On the Import JavaScript Libraries panel, check the box next to any installed Ajax libraries you want included
  5. If you want Jaxer support in your project, on the Jaxer Support panel check the desired features and click Next
  6. On the HTML Preview Settings screen, define the custom preview settings needed for your project and click Next
  7. Click Finish

Note: If the support plugin for the library you want to use is not yet installed in Studio, click the Install JavaScript Libraries button to add it.
