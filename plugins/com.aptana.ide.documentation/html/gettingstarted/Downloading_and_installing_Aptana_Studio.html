Downloading and installing Aptana Studio [edit]

This page describes how to download and install the Aptana IDE if you do not already have Eclipse installed.

  • Eclipse users: download and extract the appropriate installer for Aptana at Plugging Aptana into an existing Eclipse configuration
  • Standalone Aptana Studio users: Complete the instructions on this page

Prerequisite: Ensure your machine meets the Aptana System Requirements.

Pro Users: After completing the base installation, see Upgrade to Aptana Studio Pro to finish.

Installing Aptana Studio

To download and install Aptana Studio:

  1. Go to the downloads page to download the installer or zip file for your operating system
  2. Launch the downloaded installer

On Windows Vista you may need to set the Run this program as an administrator property to true.

Uninstalling Aptana Studio

Windows users:

  1. In Windows, go to Start > Control Panel > Add or remove programs...
  2. Select Aptana Studio from the list of programs to remove
  3. Click the Change/Remove button and follow the prompts to complete the uninstall

Mac users:

  1. In your Finder, go to the parent folder of the folder where you have Aptana installed (generally Applications)
  2. Drag the Aptana Studio folder to the Trash
  3. If prompted, confirm that you want to uninstall Aptana Studio
