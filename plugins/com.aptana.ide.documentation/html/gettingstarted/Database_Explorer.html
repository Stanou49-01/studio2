The Database Explorer plug-in to allow you to easily access your databases and run SQL queries and was added to Aptana Studio in version 1.2.

Opening the Database Explorer Perspective

x

  • Click the Database Perspective button in the upper right of your workbench
  • Use the Open Perspective menu and select Other... > Database.

x

Views

x The Database Perspective contains three additional views for accessing and querying your databases:

  • DB Tree View
  • SQL History View
  • SQL Execute View
