About Prototype [edit]

This page contains an overview about using the Prototype JavaScript framework.

Prototype is a popular JavaScript framework for building dynamic web applications. Prototype includes the following features:

  • Toolkit
  • Class-based development
  • AJAX library

See http://prototype.conio.net to download and learn more about Prototype. Have also a look on sergiopereira.com for comprehensive documentation.
