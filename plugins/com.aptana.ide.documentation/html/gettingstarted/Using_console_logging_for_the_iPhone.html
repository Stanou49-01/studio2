Using console logging for the iPhone [edit]

This Help topic describes how to use console logging for an iPhone project.

Introduction

Use the console log for the iPhone much in the same way that you would use console logging for any of your other web applications. The console can display values of variables, evaluate expressions, and other similar tasks for an application that you are previewing on your iPhone. (See Viewing a project on your iPhone) for details.)

Instructions

To use the console log with your iPhone:

  1. Open the Console View if it is not already open:
    • From the Window menu, select Show View > Console to open the Console View.
  2. Within your application, call console.log at the place where you want to log.
  3. View the application on your iPhone (see Viewing a project on your iPhone), and interact with the application to see the messages logged.

To see a demonstration of logging with the iPhone, see the iPhone screencast: http://www.aptana.tv/movies/iphone_screencast2/iphone_screencast2.html
