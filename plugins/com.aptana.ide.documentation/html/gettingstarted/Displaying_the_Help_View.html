Displaying the Help View [edit]

This page explains what the Help View is and how to display it.

Contents

  • 1 Introduction
  • 2 Instructions
  • 3 Get help topics to open in an external browser
  • 4 Related Topics

Introduction

The Help View provides context-sensitive links to Help topics related to the part of the workbench that you are currently working in.

Instructions

To display the Help View:

Click the Help button in the corner of any Aptana view.

OR

Press the F1 key to get context-sensitive Help about the current selection.

OR

  1. From the Window menu, select Show View > Other... to open a Show Views pop-up window.
  2. On the Show Views pop-up window, expand the Help folder.
  3. Select the Help View and click OK.

Aptana opens the Help View (shown below) at the right side of your workbench.

Get help topics to open in an external browser

If you would like your help topics to open in an external browser, follow these instructions:

  1. Go to "Window -> Preferences -> Help".
  2. Check "Use external browser".
  3. Click the shortcut link to "Web Browser" or go to "Window -> Preferences -> General -> Web Browser".
  4. Select "Use external Web Browser".
  5. Click "Ok".
