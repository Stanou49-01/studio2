What

x Cloud deployment is how you take a project in Aptana Studio to a live application on the Aptana Cloud. x

When

  • You're ready to deploy your project to the cloud

How

x To deploy a project on the Aptana Cloud, select Aptana -> My Aptana. On the My Aptana screen, click the "My Cloud" tab. Your local projects will be listed. Projects associated with an Aptana Cloud site will have a "synchronize" button next to them. If not, a "deploy to cloud" button will be shown.

Clicking the "synchronize" button will launch the smart sync window. See Using Smart Sync for details.


Clicking the "deploy to cloud" button will launch the deployment wizard.

Deployment

x This page will walk you through the deploy to cloud wizard screen by screen. x

When you complete a step, click the "continue" button in the bottom right of the screen.


If you have to make a change to a previous screen, click the "back" button in the bottom left of the screen.


At any time in the deploy to cloud wizard, you can cancel by clicking the "cancel" button.


Use this button on the last step of the Try Now deployment wizard to complete the process sand order your new site.


x

Try Cloud Wizard

x

Step 1: Site Name

x

Site Name
Your project will be served from a unique site in the Aptana Cloud. Enter your preferred URL in the box. Your preferred url (or site name) must be between 3 and 35 characters long, can only contain letters, numbers, and dashes (-), and must start with a letter.

Your Aptana Cloud site will have a unique IP address so you can associate it with a custom domain name like example.com instead of example.aptanacloud.com.

You may also choose whether to enable Aptana Jaxer for the site, though you can also disable or enable it later through the Cloud management application.

When you've entered the name, click the "continue" button.


x

Step 2: User Info

x

User Info
Select the Aptana ID associated with this site. Options are:
  • Use an existing Aptana ID account
  • Create a new Aptana ID account

An Aptana ID is the username and password used to secure your Aptana Cloud sites. You can manage your Aptana ID at the Aptana.com web site.

When you've logged in or created an Aptana ID, click the "continue" button.


x

Step 3: Agreement

x

Services agreement
Read the services agreement. To continue deployment, you must check the I agree to the above terms button.

To finish, click the Deploy My Site button at the bottom right of the wizard.

Buy Now Wizard

x

Step 1: Site Name

x

Site Name
Your project will be served from a unique site in the Aptana Cloud. Enter your preferred URL in the box. Your preferred url (or site name) must be between 3 and 35 characters long, can only contain letters, numbers, and dashes (-), and must start with a letter.

x

Step 2: Pick A Plan

x

Pick a plan
The services page lets you choose the plan size of your Aptana Cloud site. You can use the slider to increase the size of your cloud. Your total monthly cost is calculated at the bottom of the screen.

Choices are:

  1. 256 MB ram with 5 GB disk space
  2. 512 MB ram with 10 GB disk space
  3. 1024 MB ram with 15 GB disk space
  4. 2048 MB ram with 25 GB disk space

The size of your plan will affect the average daily cost. Adjusting the slider will show you the daily average cost.

When you are happy with your service selections, click the "continue" button.


x

Step 3: Plan Options

x

Plan Options
Select your support options, team options, and cloud configuration.


x

Step 4: User Info

x

User Info
Select the Aptana ID associated with this site. Options are:
  • The account you are currently logged in with. When choosing this option, you will be prompted for the existing user's password again, for security.
  • A different existing Aptana ID account
  • Create a new Aptana ID account

An Aptana ID is the username and password used to secure your Aptana Cloud sites. You can manage your Aptana ID at the Aptana.com web site.

When you've logged in or created an Aptana ID, click the "continue" button.


x

Step 5: Payment Info

x

Billing info
Enter your Billing Information on this screen and click the "continue" button. Enter a coupon code if you have one.


x

Step 6: Agreement

x

Services agreement
Read the services agreement. To continue deployment, you must check the I agree to the above terms button. Then click the "continue" button.


x

Step 7: Confirm

x

Confirm and place order
The last step before deployment is a confirmation screen. You can view all of the information you have entered and settings you have selected for your site and confirm them. The settings you must confirm are:
  • Project Info
  • Your Info
  • Billing Info
  • Total Monthly Cost


If any information is incorrect, you can use the back button to return to a previous step in the wizard. If everything looks good, finish the wizard by clicking the "place order" button.


x

After Placing Your Order

x

Provisioning Progress Screen

x

Provisioning your site
After clicking the "place order" button, you will see the provisioning progress screen. This screen shows you a progress bar and will display the current tasks and messages as your project is deployed on the Aptana Cloud. The steps include:
  • Verifying your Order Information
  • Provisioning Your Site
  • Starting-Up the Aptana Cloud Environment


x

Your site is running!

x

Your site is running
When your site has finished provisioning, you can use it right away. To complete the deployment process, you need to synchronize your files. See Using Smart Sync for details. You can also manage this project in My Cloud. See My Cloud - Overview for details.
