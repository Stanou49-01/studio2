Adding a site connection [edit]

This page describes how to add a site connection between a remote FTP or SFTP site and a folder or project on your local hard drive.

  1. On the Sync Manager View (which may be hidden behind the Code Assist Profiles View), click the Add Site Connection button to open a Create Site Connection window.

    You can also right-click on a folder in your File View or a project in Project view and select Synchronize > Create New Synchronize Connection from the context menu.

  2. On the Create Site Connection window, define your connection.
    1. In the Connection name field, type a name for the connection.
    2. From the Local Path drop-down list, select File or Project to specify using either the File View or Project View as your local source.
    3. For the local path, click the Browse button to browse to the folder or project that you want to sync.(If you right-clicked a folder in File View to add the connection, this will automatically be filled in for you.)
    4. From the Select remote server... drop-down list, select the remote site that you want to connect to. If you do not see the remote site that you want, select one of the following options:
      • Add new Project Shortcuts... - Access your Project View.
      • Add new SFTP... - Add a new SFTP connection.
      • Add new FTP... - Add a new FTP connection.

      (See Adding an FTP site to learn how to add an FTP site.)

    5. Click OK.

      6. Aptana adds your new connection to the Sync Manager and to the corresponding folder in the File View.
