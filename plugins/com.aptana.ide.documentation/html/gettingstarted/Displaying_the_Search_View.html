Displaying the Search View [edit]

This page describes what the Search View is and how to display it.

Introduction

The Search View displays all of your retained results from your most recent search.

Instructions

To display the Search View, click the Search button and perform a search. (See Searching Aptana for more information on performing searches.) Aptana opens a Search View and displays the results in the View (shown below).

To clear the contents of the Search View, perform a new search or click the Remove All Matches button .
