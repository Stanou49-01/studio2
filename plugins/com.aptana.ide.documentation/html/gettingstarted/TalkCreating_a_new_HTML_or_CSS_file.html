If you want to be able to create a new HTML (or other) file without using the wizard, you'll need to be in File View instead of Project View. I'm betting that your issue stems from the fact that File View loses its functionality when you are using Aptana as a plugin for Eclipse 3.2. This is a known issue. See Release Notes Archive#Known_Issues_2. HTH.
