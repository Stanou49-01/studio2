Reporting an Aptana bug or requesting a feature [edit]

This page describes how to report a bug or request a feature in Aptana.

Introduction

If you find an application bug, please submit a bug report so that we can continue to improve Aptana.

Additionally, to help us plan and prioritize which features should be included in upcoming Aptana releases, you can submit a feature request if there is a feature that you would like to see that is not in the current release.

Use http://support.aptana.com/asap (ASAP) to submit both bugs and feature requests.

Instructions

You can report a bug or request a feature either from Aptana Studio, or by going directly to http://support.aptana.com/asap.

To report a bug or request a feature from Aptana Studio:

  • From the Help' menu, select Aptana Troubleshooting > Submit Aptana Bug or Feedback... to go to the Create 'New Ticket page in ASAP.
  • If you do not have an account in ASAP, you will need to create one. This account is currently different than other Aptana accounts.
  • Fill out the Create New Ticket window to the best of your ability:
    • Select the correct product you are filing the bug against
    • From the Type drop-down list, select whether you are submitting a bug or enhancement request.
    • In the Description text area, type as complete a description as you can for your bug or feature. If you are submitting a bug, include the steps to reproduce the bug.
  • Click the Submit button to submit your bug or feature.

Your bug or feature request will be entered into the ASAP system for the Aptana development team.

Helpful Tips

When submitting a bug, below are some questions that are helpful to the development team include answers for:

  • Search the bug database first for a pre-existing bug. It's much more helpful if we have one bug ticket to cover a particular issue. It allows us to keep everyone up to date as to status and progress more efficiently.
  • What version/build # of Aptana Studio are you using?
  • What Operating System are you using?
  • Is it a standalone version, or plugged into Eclipse? If Eclipse, which version of Eclipse?
  • How did you install the product? Is it a re-installation?
  • Did this item/feature work previously in a past version? When did it stop working?
  • Steps to reproduce the problem. Please be as exact as possible. Code samples, screenshots and screencasts are great! We love them.
  • Include a log file if at all possible. Depending on what happened, it will be in one of two places.
    • If Aptana Studio crashed entirely and you saw a "JVM terminated" dialog, look here: Reporting an error or crash in Aptana
    • Otherwise, look here: Accessing your Aptana log
  • Finally, please only file one issue per bug report. Multiple tickets for multiple bugs makes it much easier to handle your issue.
