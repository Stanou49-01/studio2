Editing a Snippet [edit]

This page describes how to edit a snippet in the Snippets View.

Introduction

After you have created a snippet, you can edit it later on if you need to. You can also edit any of the default snippets that are included with Aptana.

Instructions

To edit a snippet that inserts text, such as the "Insert <table>" snippet:

  1. In the Snippets View, right-click the snippet that you want to edit.
  2. From the context menu, select Edit Snippet to open the snippet in the Aptana Editor.
  3. Make your changes to the snippet file.
  4. Save your changes.
