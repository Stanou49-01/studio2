About the Aptana toolbar [edit]

This page describes the buttons on the Aptana toolbar.

Use the Aptana toolbar at the top of your Workbench as a shortcut to access common tasks. The list below describes the functionality for each button.

  • - Use the New button to create new projects, JavaScript, HTML, or CSS files.
  • - Use the Save button to save all changes in your workspace.
  • - Use the Print button to print a file.
  • - Use the Reload button to rebuild the scripting environment and reload your scripts.
  • - Use the Run button to preview your project in a web browser.
  • - Use the Search icon to search for a string in your code.
  • - Use the Last Edit Location button to jump to the last place you made a change in your code.
  • - Use the Next/Previous buttons to navigate to other open files in your project.
