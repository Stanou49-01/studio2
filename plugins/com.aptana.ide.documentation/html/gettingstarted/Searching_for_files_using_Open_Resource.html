Searching for files using Open Resource [edit]

This page explains how to use the Open Resource feature to quickly search for files in your workspace.

Introduction

Use the Open Resource file to quickly find files in your workspace whose file names contain a string that you specify.

Instructions

To use Open Resource to search for files:

  1. From the Navigate menu, select Open Resource... to open an Open Resource pop-up window.

    Note: You can also use the shortcut ctrl+shift+R to open the Open Resource pop-up window.

  2. In the Select a Resource text box, type a string that you want to search your file names for. You can use "?" as a wildcard to search for any character, or "*" as a wildcard to search for any string.

    As you type, Aptana displays a list of files containing the string in the Matching Resources box and their corresponding folders in the In folders box.

  3. In the Matching Resources box, select the file that you want to open.
  4. Click OK.

Aptana opens the file that you selected in your editor.
