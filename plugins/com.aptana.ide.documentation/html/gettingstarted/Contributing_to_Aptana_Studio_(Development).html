Contributing to Aptana Studio (Development) [edit]

Aptana Studio is open source software, available under a dual GPL (v3) and APL license. (See our License FAQ to learn more about our license).

We welcome collaborations with the developer community and have an entire developer wiki set up to help you get started.

Contributing to Aptana Studio development:

  1. Familiarize yourself with the Aptana Studio Developer Wiki
  2. Install one of the Subversion plug-ins to access our source code

With an SVN plug-in installed, you can download the Aptana Studio source code.
