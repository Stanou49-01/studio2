About Aptana Snippets [edit]

This page gives an overview of the code Snippet feature in Aptana.

Introduction

Use the code Snippet feature in Aptana to quickly insert small pieces of code into your HTML, CSS, JavaScript, and XML files. Snippets can save you time and hassle of typing out bits of code that you frequently use by adding them into your files by simply double-clicking the name of the Snippet.
