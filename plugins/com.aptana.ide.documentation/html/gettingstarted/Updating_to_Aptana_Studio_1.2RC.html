Updating to Aptana Studio 1.2RC [edit]

These instructions are for people who have Aptana Studio 1.1 already installed. If you do not already have Aptana Studio 1.1 or later download Aptana Studio and follow the directions on that page.

These instructions are done within Aptana Studio.

  1. Navigate to Window > Preferences and select Install/Update
  2. Set the Policy URL to: http://update.aptana.com/update/next/3.2/policy.xml
  3. Press OK
  4. Navigate to Help -> Check For Aptana Updates Now... and follow the wizard's instructions
  5. Pro Users must repeat "Navigate To Help -> Check For Updates" after restart to enable their Pro functionality.
