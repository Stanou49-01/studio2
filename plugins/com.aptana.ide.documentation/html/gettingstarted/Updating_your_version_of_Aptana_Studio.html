Updating your version of Aptana Studio [edit]

Aptana Studio automatically checks for updates when it starts up and let you install any found.

You can also manually check for updates within Studio by choosing Check for Aptana Updates from the Help menu.

If there are none available, in either case, Aptana will display a no updates message.

  1. From the Help menu, select Check for Aptana Updates Now...
  2. If updates are available, the Update Manager will list them
  3. Check the updates you wish to install and click Next
  4. Choose the option for I accept the terms of the license agreement and click Next
  5. Click Finish
  6. You may need to agree to install an unsigned plugin, this is due to a quirk in Eclipse plugin packaging which we are working on correcting
  7. Afterwards you'll be prompted to quit and restart Aptana Studio
