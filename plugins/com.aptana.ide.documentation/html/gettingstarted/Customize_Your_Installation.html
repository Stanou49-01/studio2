Customize Your Installation [edit]

When you launch Aptana Studio the Customize Aptana Studio wizard will automatically run so you can easily install additional plugins, including:

The Wizard
  • Web Application Platforms:
    • PHP
    • RadRails
    • Jaxer
    • Pydev
  • Ajax Libraries
    • jQuery
    • Yahoo! User Interface
    • Dojo
    • ExtJS
    • Prototype/Scriptaculous
    • Mootools
  • Mobile
    • iPhone support
    • Nokia WRT
  • Desktop
    • Adobe AIR
  • Others
    • SCM Plugins
      • Git
      • Subversive
      • Subclipse
      • P4WSAD (Perforce)
      • CVS
    • Internationalization Support
    • vi Support

Select the plugins you need and click Install to add them to your Studio. You'll be able to install plugins later on as well (using Install Additional Features from the Aptana menu) so don't worry if you miss one here.

Note: If you check the Do not show again box this wizard will not run on subsequent launches.
