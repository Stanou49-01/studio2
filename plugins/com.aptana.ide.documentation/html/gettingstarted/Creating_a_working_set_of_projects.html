Creating a working set of projects [edit]

This page describes how to create a working set of your projects in Aptana.

Introduction

If you work on many different projects in Aptana (or Eclipse), you may not want to have all of your projects visible in the Project View all of the time. You can use a "working set" to create a filter so that only the projects that you want to see are visible. Working sets allow you to create "sub-projects" within a project.

Instructions

To create a working set:

  1. On the Project menu in Aptana (or Eclipse), make sure that Build Automatically is NOT checked.
  2. From the Project menu, select Build Working Set > Select Working Set... to open a Select Working Set pop-up window.
  3. On the Select Working Set pop-up window, click the New... button to open a New Working Set pop-up window.
  4. On the New Working Set pop-up window, select Resource and click the Next button to go to the Resource Working Set screen.
  5. In the Working set name field, type a name for your new working set.
  6. On the list of projects, check the boxes for all of the projects that you would like to include in this working set.
  7. Click the Finish button.
  8. Click OK.
  9. On the Project View, click the down-facing white arrow in the upper-right of the Project View toolbar area, and select your working set.

The Aptana Project View refreshes to show only the projects that you selected for your working set.
