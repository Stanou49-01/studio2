Using RadRails Views [edit]

This page gives an overview of the various RadRails Views.

Introduction

While Eclipse, Aptana, and RadRails share many Views and many of these Views have similar functionality across IDE's and plug-ins, RadRails has several Views that are specific to RadRails or that have slightly different functionality in RadRails. This page lists and gives links to these RadRails-specific Views.

Reference

The following Views are either unique to RadRails or contain functionality that is unique to RadRails:

  • Console View
  • Servers View
  • Generators View
  • Rails Plugins View
  • RegExp View
  • Rake Tasks View
  • Rails Shell View
