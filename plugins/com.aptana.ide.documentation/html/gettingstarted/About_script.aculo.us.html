About script.aculo.us [edit]

This page gives an overview about using the script.aculo.us JavaScript library.

script.aculo.us is a set of JavaScript libraries that help you build dynamic, interactive web applications. script.aculo.us currently supports the following features:

  • Visual effects
  • Drag-and-drop
  • Controls

See http://script.aculo.us to learn more about and download script.aculo.us.
