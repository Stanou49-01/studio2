Navigating open files with Next Editor or Previous Editor [edit]

This page explains how to use the Next Editor or Previous Editor features to tab through your open files.

Introduction

In Aptana, instead of using alt+tab to navigate through a series of open files, use the keyboard shortcuts ctrl+F6 and ctrl+shift+F6 to navigate between files that you have open in the editor.

Instructions

  • To go to the next file open in your editor, use the shortcut key combination of ctrl+F6 or click the Next button in the toolbar.
  • To return to the previous file open in your editor, use the shortcut key combination of ctrl+shift+F6 or click the Previous button in the toolbar.

Note: You can also access this feature from the Window menu by selecting Navigation > Next Editor or Navigation > Previous Editor.
