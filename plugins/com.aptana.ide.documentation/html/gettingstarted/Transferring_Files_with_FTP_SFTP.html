Aptana Studio's FTP and SFTP tools enable you to transfer files between your hard drive and a remote web site.

Setup

  1. Create a connection to your remote web site (Adding an SFTP/FTP site).
  2. Create a site connection between the SFTP/FTP site and the folder or Aptana project on your hard drive (Adding a site connection).

You can now transfer your files between locations using the Sync Manager or with drag and drop between your remote site and any folder in the File View.

Note: SFTP support is only available in the Pro Edition of Aptana Studio. An older, no longer supported SFTP plugin that may be used with Community Edition is available on our Plugins page.
