Adding an FTP site [edit]

This page describes how to create a new web site connection for Aptana's FTP tool.


Prerequisite: Set up a connection to your remote web site (Adding a site connection).

Instructions

To create a new site connection:

  1. Click the File tab to display the File View.
  2. On the File View, right-click either SFTP (secure FTP) or FTP, depending on which type of connection you want to create, and select Add New (S)FTP Site....

    Aptana Studio will open a Site Configuration window.

  3. On the Site Configuration window, define the information for the site.
    1. In the Site name field, type a name for the new remote site.
    2. In the Server field, type the URL for the site that you will be connecting to (e.g. admin.myWebSite.com).
    3. If you want to connect to a subfolder under the root of the remote site, type the folder path in the Remote Path field.
    4. If you want to use a port other than "21", type the port in the Port field.
    5. In the Username and Password fields, type your username and password for the server.
    6. If you want to test that your username and password work for this particular set up, click the Test button.
    7. If you want to save your username and password settings, check the Save box to the right of the Password field.
    8. If you work in a team environment and could possibly be uploading files that could overwrite others' changes, check the Calculate server/client time offset automatically box as a precaution.
    9. Click OK.

    4. Aptana Studio will create the new connection for you.

    You can modify the connection from your File View. To change the connection, right-click the connection and select Properties from the context menu.

    To transfer files to the site or download files from the site, see Synchronizing your files or to edit files directly without using FTP, see Editing a file from a remote site.
