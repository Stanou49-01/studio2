Configuring your JDBC drivers [edit]

What

You configure the JDBC drivers for your database through preferences.

Prerequisites: The name and connection string for each driver.

How

Connection string preferences
To configure your JDBC drivers:
  • From the Window menu, select Preferences... > Database > Connection String
  • Add: Click the New button and enter the driver name and connection string
  • Edit: Select the driver and click the Edit button. This allows you to change either the driver name, connection string, or both
  • Delete: Select the driver and click the Delete button
