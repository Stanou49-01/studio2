Mac OS X Installation Tips [edit]

This page is a compilation of tips for configuring and using RadRails on a Mac OS X system.

Configuration tutorial

For help with setting up RadRails on Mac OS X, see Hivelogic's comprehensive article and tutorial:

Building Ruby, Rails, Subversion, Mongrel, and MySQL on Mac OS X

Locomotive users

If you are using Locomotive with Mac OS X, look in the following directory to help configure your RadRails setup:

/Applications/Locomotive2/Bundles/standardRailsSept2006.locobundle/i386/bin/rails

Your ruby, rake, and mongrel_rails files should be contained here.

If you need more help, read the step by step tutorial for Ruby on Rails Development on Mac OS X.

You can also find more tips at http://iloverails.blogspot.com/2007/02/getting-radrails-to-work-with.html.
