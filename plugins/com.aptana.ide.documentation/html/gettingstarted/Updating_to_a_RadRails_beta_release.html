Updating to a RadRails beta release [edit]

This page explains how to configure an existing RadRails installation to retrieve the latest beta release of radRails.

Introduction

In some instances, RadRails developers will decide to install the beta version of a RadRails release to use bleeding edge features before the build is fully tested.

Instructions

Setting the policy URL

Click on Window and select Preferences

Set the policy URL to http://update.aptana.com/update/rails/beta/3.2/policy.xml

Updating the RadRails Installation

Click on Help and select Check for Aptana Updates

Accept all the default dialogs and say yes to the restart dialog.
