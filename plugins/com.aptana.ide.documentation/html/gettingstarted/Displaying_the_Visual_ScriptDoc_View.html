Displaying the Visual ScriptDoc View [edit]

The Visual ScriptDoc View provides a visual navigation of JavaScript code in your projects, written by you or 3rd party libraries. If the code is documented with ScriptDoc comments the view also provides function level information, such as definitions and return values.

To open this view select View -> Visual ScriptDoc' from the Aptana menu.

You can click on the function or object names to navigate through the source code tree, use the Right and Left arrow buttons to navigate forward and backwards through the view and reload the contents by right clicking on it and selecting Reload from the context menu.
