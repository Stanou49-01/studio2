Installing and running Aptana on Vista [edit]

The main difference in setting up standalone Aptana Studio to run on Microsoft Windows Vista from other versions of Windows is that you must set Aptana to run as Administrator to be able to install or update new plugins.

Instructions

  1. Right-click the Aptana shortcut, and select Properties from the context menu.
  2. Go to the Shortcut tab.
  3. Click the Advanced button
  4. Check the Run this program as an administrator box.
