Getting started with AIR development [edit]

The Aptana AIR plugin for Aptana Studio enables you to quickly create, debug and publish Adobe AIR applications.

  • Install the AIR plugin for Aptana Studio
  • Install the Adobe AIR Runtime by choosing Help -> Install Adobe AIR Runtime...
  • Configure the included AIR runtime
  • Configure an alternative AIR runtime for when you need to use a different SDK
  • Create your AIR project using the Aptana Studio New Project wizard
  • Run or Debug your AIR application
  • Package and publish your AIR application
  • Sign your AIR project to add a digital certificate to your AIR application
  • Configure Certificates Preferences
  • Deploying Adobe AIR Applications to Aptana Cloud
