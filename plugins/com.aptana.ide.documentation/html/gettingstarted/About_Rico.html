About Rico [edit]

This page contains an overview about using the Rico JavaScript library.

Rico is an open source JavaScript library for creating rich internet applications. Rico includes support for the following features:

  • AJAX support
  • Drag and drop
  • Cinematic effects
  • Behaviors

See http://openrico.org to learn more about and download Rico.
