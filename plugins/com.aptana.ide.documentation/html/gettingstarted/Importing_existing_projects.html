Importing existing projects [edit]

This help topic describes how to import existing projects (e.g. from an old workspace) to your current workspace.

Introduction

If you change workspaces or install a new version of Aptana, such as to transition from Milestone 7 to Milestone 8a, you can import your existing projects into your new workspace.

Instructions

To import existing projects into a different workspace:

  1. From the File menu, select Import... > General > Existing Projects into Workspace and click the Next button.
  2. Choose the Select root directory option, and click the Browse button to browse to the root directory of your old workspace or other directory where your projects are located.

    A list of projects in the specified directory appears in the Projects list.

  3. On the Projects list, make sure that the boxes next to all of the projects that you want to import are checked.
  4. If you want to copy your projects into the new workspace, check the Copy projects into workspace box.
  5. Click the Finish button to import your projects.
