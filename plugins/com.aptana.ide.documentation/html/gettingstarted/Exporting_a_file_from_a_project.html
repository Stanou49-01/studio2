Exporting a file from a project [edit]

This page describes how to export a file from a project.

Introduction

You can export files from an Aptana project into another directory or archive.

Instructions

To export a file:

  1. In the Project View, right-click the project that you want to export files from and select Export... from the context menu.
  2. Select an export destination, and click the Next button.
  3. Check the box(es) next to the file(s) that you want to export.
  4. In the To directory text box, browse to the directory that you want to export the file(s) to.
  5. Click the Finish button.

Aptana exports the specified file(s) to the specified directory.
