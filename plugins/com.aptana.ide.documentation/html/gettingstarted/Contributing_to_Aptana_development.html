Contributing to Aptana development [edit]

This page describes how to get started with developing for Aptana Studio.

Introduction

Aptana is open source software, available under a dual GPL (v3) and APL license. (See our [http://www.aptana.com/legal/ License FAQ to learn more about our license).

We welcome contributions from the developer community and have an entire developer wiki set up to help you get started:

http://aptana.com/dev/index.php/Main_Page

Instructions

To get started with contributing to Aptana development:

  1. Familiarize yourself with the Aptana developer wiki:

    http://aptana.com/dev/index.php/Main_Page

  2. Install one of the Subversion plug-ins to access our source code:

    http://aptana.com/dev/index.php/Getting_started_with_Subversion

With an SVN plug-in installed, you should be able to browse and download the Aptana source code.
