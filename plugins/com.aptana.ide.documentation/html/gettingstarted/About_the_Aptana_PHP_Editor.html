About the Aptana PHP Editor [edit]

This page gives an overview of the Aptana PHP Editor.

Introduction

Use the PHP Editor to edit PHP files or files with PHP content for your web development projects. The PHP Editor provides colorization, folding, and Code Assist for your PHP files.
