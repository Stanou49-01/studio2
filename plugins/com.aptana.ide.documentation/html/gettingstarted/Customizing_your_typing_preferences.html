Customizing your typing preferences [edit]

This page describes how to customize your preferences for typing in the various Aptana editors.

Introduction

You can modify your preferences for typing in the various Aptana Editors to modify how things like quote matching and HTML tag completion work.

Instructions

To customize your code typing preferences:

  1. From the Window menu, select Preferences... to open a Preferences window.
  2. On the tree on the left, navigate to Aptana > Editors, expand the Editor that you want to change the code typing for, and select Typing.

  3. Modify your typing preferences to fit your needs.
  4. Click OK.
