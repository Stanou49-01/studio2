Task List (Ruby development) [edit]

This page gives an overview of the Ruby Task list.

Introduction

The task list shows you a list of tasks in your workspace. Tasks can be related to keywords (tags) in ruby comments.

Define your tags that will add a task to the task list in Preferences... > Ruby > Task Tags.

The keywords TODO and FIXME are defined as task tags by default.
