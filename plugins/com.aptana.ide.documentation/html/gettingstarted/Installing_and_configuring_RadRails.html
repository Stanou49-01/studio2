Installing and configuring RadRails [edit]

This page gives an overview of how to install and configure RadRails.

Introduction

This Help topic helps you install and configure RadRails on your system.

Instructions

To install and configure RadRails:

  1. Install RadRails on your computer from the RadRails download page.
  2. Configure RadRails to work with your Ruby configuration.
