Running two versions of Aptana Studio [edit]

Install Studio normally for the first version

For the second install

  1. Run installer again but change the installation location to a different folder
  2. Before launching this instance, edit config.ini in the installation folder
  3. Find the last section of the file, starting with # The default workspace location and change each of the three locations in it (the values will differ based on your OS, but in this example we added the word Beta to the default folder name):
    • osgi.instance.area.default=@user.home/Documents/Aptana Studio Beta
    • osgi.configuration.area=@user.home/Library/Application Support/Aptana/Aptana Studio Beta/configuration
    • osgi.user.area=@user.home/Library/Application Support/Aptana/Aptana Studio Beta/user
  4. Save your changes

Note that you will only be able to run one instance at a time but you can switch as needed.
