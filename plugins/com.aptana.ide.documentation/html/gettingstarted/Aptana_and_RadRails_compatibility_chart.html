Aptana and RadRails compatibility chart [edit]

This page explains the compatibility requirements for the various versions of Aptana, RadRails, and Eclipse.

1.0.0 Requirements

Requires Aptana 1.1+ standalone or Aptana 1.1+ plugins plus Eclipse 3.2 or 3.3.

0.9.x Requirements

Requires Aptana 1.0+ standalone or Aptana 1.0+ plugins plus Eclipse 3.2 or 3.3.

0.8.x Requirements

Requires Aptana 0.2.8+ standalone or Aptana 0.2.8+ plugins plus Eclipse 3.2.

0.7.2 Requirements

RadRails 0.7.2 standalone is built on Eclipse 3.1 and includes RDT 0.8.0. There are no requirements.

RadRails 0.7.2 plugin requires Eclipse 3.1 and RDT 0.8.0 (do not use RDT 0.9.0RC or higher).
