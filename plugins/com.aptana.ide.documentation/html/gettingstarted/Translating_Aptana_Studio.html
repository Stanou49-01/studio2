Translating Aptana Studio [edit]

This page describes how to get started with translating Aptana from English to another language.

Introduction

If you are a translation volunteer, we have a special web site up for you to help you translate Aptana to other languages. Follow the instructions below to get started with translating Aptana.

Instructions

  1. Go to the Aptana translation web site:

    http://translate.aptana.com

  2. Login to the web site using your username and password.
  3. Click the Start button to display the list of Aptana projects.
  4. From the Select Language drop-down list, select your language.
  5. On the main part of the page, select an Aptana project from the list
    • Aptana FTP Synchronization Provider
    • Aptana Synchronization Core
    • Aptana Regular Expressions
    • Aptana IDE
    • Aptana Parsing
    • Aptana Lexer
    • Aptana Unified Editor
    • Aptana XML Editor
    • Aptana SQL Editor
    • Aptana PHP Editor
    • Aptana JavaScript Editor
    • Aptana HTML Editor

    The page will display a list of project entries that are available for translation.

  6. From the list of entries in the left pane, select the entry that you would like to translate.
  7. In the Translation field, type in the translation for the entry.
  8. Click the Save Translation button to save your translated text.
  9. Repeat Steps 6-9 for each entry that you would like to translate.
