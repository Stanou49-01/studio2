Ruby Debug Perspective [edit]

This page gives an overview of the Ruby Debug Perspective.

Introduction

The Ruby Debug Perspective is designed for debugging your Ruby program. It includes an editor area and the following views:

  • Ruby Debug Views
  • Ruby Breakpoints View
  • Ruby Expressions View
  • Ruby Variables View
  • Ruby Display View
  • Ruby Outline View
  • Ruby Console View
