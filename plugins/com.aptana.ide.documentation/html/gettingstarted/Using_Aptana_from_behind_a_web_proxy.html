Using Aptana from behind a web proxy [edit]

This page describes how to use Aptana if you are behind a web proxy, including authenticating ones.

This support is considered Beta; please report any feedback you have at http://support.aptana.com/asap/secure/Dashboard.jspa.

Introduction

Aptana connects to the web for news, documentation and code updates. This guide will help you configure Aptana if you are behind a web proxy.

Instructions

To configure proxy support:

  1. Configure your proxy settings.
    1. From the Window menu, select Preferences... > Aptana > Internet Proxy Settings.
    2. Check the Enable proxy box.
    3. In the Proxy host and Proxy port fields, specify the IP address and port number of your proxy server.
    4. If you have authentication enabled, check the Enable proxy authentication box and type in your proxy username and password.
    5. Click OK to apply your settings.
  2. Restart Aptana.

    Aptana should now use the proxy server to connect.
