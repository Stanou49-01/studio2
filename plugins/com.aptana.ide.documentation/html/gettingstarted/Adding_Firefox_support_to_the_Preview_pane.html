Adding Firefox support to the Preview pane [edit]

All current versions of Aptana Studio (1.1 or later) should included built in support for Firefox in the Preview Pane.

If you are using an earlier version of Aptana Studio, we highly recommend backing up your system (e.g. workspace, preferences, etc), uninstalling your old version, and downloading and installing the latest version from our Products page.

http://www.aptana.com/studio/
