About AFLAX [edit]

This page contains an overview about using the AFLAX library.

The AFLAX library allows developers to use JavaScript with Flash together to rich, dynamic web applications that take advantage of Flash's features without having to learn Flash. Use AFLAX to take advantage of most of the Flash 8 platform, including drawing commands, filters, and video.

See http://www.aflax.org to download and learn more about AFLAX.
