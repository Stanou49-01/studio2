Adding a web browser configuration to your Favorites [edit]

This page describes how to add a web browser configuration to your browser preview Favorites menu.

Introduction

If you create several browser configurations for previewing web pages, you can always access them by clicking the arrow to the right of the Run button . However, if you find that you use certain configurations more frequently than others, you can add those configurations to the Favorites menu, which will make them always appear on the top part of the Run menu.

(See Setting up your web browser configuration to learn how to set up different configurations if you need help.)

Instructions

To add a browser configuration to your favorites:

  1. Click the arrow to the right of the Run button and select Organize Favorites... to open an Organize Run Configuration Favorites pop-up window.
  2. On the Organize Run Configuration Favorites pop-up window, click the Add... button to open an Add Run Configuration Favorites pop-up window.
  3. On the Add Run Configuration Favorites pop-up window, check the boxes next to any configurations that you want to add to your favorites.
  4. Click OK to close the Add Run Configuration Favorites pop-up window.
  5. Click OK to close the Organize Run Configuration Favorites pop-up window.

Aptana adds the checked configurations to your favorites. If you click the arrow to the right of the Run button , you can see that the specified configurations are now at the top of the menu.
