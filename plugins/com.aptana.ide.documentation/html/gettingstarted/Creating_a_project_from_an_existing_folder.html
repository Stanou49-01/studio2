Creating a project from an existing folder [edit]

This page describes how to create a project from a folder on disk.

Introduction

You can import a folder as a project into Aptana. It does not move or alter the files in any way, but allows you to use SVN, CVS or other source control systems with your files..

Instructions

To convert a folder to a project:

  1. In the File View, right-click the folder that you want to convert.
  2. Choose Promote to Project...
  3. Give the project a name. The name must be unique across all projects in your "workspace"
  4. Click "OK". The project appears in your workspace.

Alternately, you may choose the similar wizard in File > Import...
