Contributing to Aptana Studio Wiki (Documentation) [edit]

The Aptana wiki is a dynamic, community-based online documentation system to help web developers get the most out of the Aptana IDE.

This page describes how to use and contribute to the online documentation wiki for Aptana Studio.

Wiki Guidelines

The wiki is your main source for online help about Aptana Studio. Although there are no restrictions on reading the wiki documentation online, you must create an account and log in to contribute the wiki and the wiki is moderated by Aptana staff.

Use the wiki for

  • Adding helpful procedures about using the product.
  • Providing additional details to clarify an existing procedure.
  • Using the Discussion page for a topic to discuss the Help topic. The Discussion pages on the Aptana wiki are there for a purpose similar to the Discussion pages on Wikipedia.
  • Making minor text corrections (e.g. typos), adding details, and clarification of explanations/instructions.

Do not use the wiki for

  • Logging bugs about the software. If you discover a bug in Aptana, please log the issue in our ASAP Issue Tracker.
  • Making significant changes or corrections to the documentation. If you find a doc bug, we need to know about it because more than one Help topic might be affected. Please log the documentation bug in ASAP Issue Tracker.
  • Making Aptana feature requests. If you have a feature request, please log the request in ASAP Issue Tracker.
  • Asking Aptana support questions. If you have a technical support question, please post it on the Aptana Forums as your question will be answered much more quickly there.
