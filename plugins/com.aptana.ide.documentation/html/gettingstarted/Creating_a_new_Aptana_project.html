Creating a new Aptana project [edit]

This page describes how to create a new Aptana project.

Introduction

If you decide that you want to group your related web development files into projects, create an Aptana project for those files to be associated with.

Instructions

To create a new Aptana project:

  1. From the File menu, select New > Project... to open a New Project window.
  2. On the New Project window, expand the Simple folder. (If you are using Aptana as a plug-in for Eclipse v3.2, expand the General folder.)
  3. Select Project.
  4. Click the Next button.
  5. In the Project Name text box, type a name for your new project.
  6. In the Referenced Projects box, if you want the new project to reference any of your other projects, check the appropriate projects.
  7. Click the Finish button.

Aptana creates the new project for you. The new project is now visible in the Project View on the left side of the screen.
