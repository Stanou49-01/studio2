Importing an existing project from Dreamweaver or HomeSite [edit]

This page describes how to import a project that you created in Dreamweaver, HomeSite, or other web development application into Aptana.

Introduction

You can import an existing project into Aptana by creating a new project and pointing Aptana to the home directory for your existing project.

Instructions

To import an existing project into Aptana:

  1. From the File menu, select New > Project... to open a New Project pop-up window.
  2. On the New Project window, expand the Simple folder.
  3. Select Project, and click the Next button.
  4. Point your new project to the folder for your existing project.
    1. In the Project Name text box, type a name for your new project.
    2. In the Project Contents section, uncheck the Use Default box.
    3. In the Directory text field, browse to the folder that contains your Dreamweaver or HomeSite project.
  5. Click the Next button.
  6. In the Referenced Projects box, if you want the new project to reference any of your other projects, check the appropriate projects.
  7. Click the Finish button.
