Synchronizing your files [edit]

This page describes how to create a new FTP sync to synchronize the files on your hard drive with a remote site.

Prerequisite: Set up a connection to your remote web site (Adding an SFTP/FTP site, Adding a site connection).

Instructions

To synchronize your files using FTP:

  1. In the Sync Manager View, double-click on your new connection or right-click on it and select Synchronize... from the context menu. (You can also right-click on the folder in the File View or a project in the Project View and select Synchronize > Synchronize... from the context menu.)

    Aptana Studio will open a Smart Sync dialog.

  2. In the Smart Sync dialog, specify how you would like to synchronize your files.
    1. Choose your synchronization direction in the Mode drop-down list:
      • Sync synchronizes both the remote and local sites so that both directories have the newest versions of each file.
      • Update -> transfers the newer files from your hard drive to the remote site.
      • Force -> transfers all the files, regardless if they are changed, from your hard drive to the remote site.
      • <- Update downloads the newer files from the remote site to your hard drive.
      • <- Force downloads all the files, regardless if they are changed, from the remote site to your hard drive.
    2. Choose the view options using the View Options: drop-down list:
      • Flat displays the items as a flat list.
      • Tree displays the items in a directory tree structure.
      • Check Show Modification Dates to display the modification times of the files on your local hard drive and the remote site in additional columns.
    3. Check the box next to either or both of the Delete... options for synchronization:
      • Delete orphaned files on <local drive> - Deletes files on your local hard drive that were not found on the remote site. This cleans up files at your local hard drive.
      • Delete orphaned files on <remote site> - Deletes files on the remote site that were not found on your local hard drive. This cleans up files at the remote site.
    4. Use the "Advanced Options" to set additional settings:
      • Use CRC check when files are less than 1 minute apart - Performs a non-timestamp based check to ensure that the newest files are used in synchronization.
      • Generate synchronization status in background- Runs the generation of the files to be synced in the background and only brings up the Smart Sync dialog when it finished.
    5. To have FTP "ignore" a file, check the box in the Skip column. Uncheck it to have the file included in the synchronization.
  3. Click the Start Sync button to synchronize your files.

    Aptana Studio will synchronize the files between the folder or project that you specified and your remote site. Both File View and Project View will refresh automatically.
