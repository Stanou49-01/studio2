Features Added to Support Adobe AIR 1.1 [edit]

On June 16, 2008 Adobe released version 1.1 of Adobe® AIR™.

What's New in Aptana's support for Adobe AIR 1.1?

To support the new features in Adobe AIR 1.1, the Adobe AIR plug-in for Aptana Studio was updated the same day as follows:

  • Added two new samples for 1.1
    • AIRMenuBuilder
    • AIRLocalizer
  • Updated the following files for new AIR projects
    • AIRIntrospector.js
    • application.xml
  • Added the following files to new AIR projects
    • AIRLocalizer.js
    • AIRMenuBuilder.js
    • AIRSourceViewer.js
  • Updated the content assist to include new 1.1 APIs
  • Updated bundled Windows and Mac SDK to 1.1
  • Updated bundle Windows Runtime to 1.1
  • Updated Mac Runtime installer

How to get the Adobe AIR 1.1 Plug-In for Aptana Studio

Existing Adobe AIR users should upgrade via Aptana Studio > Help > Check for Aptana Updates, then select the desired updates.

If you do not already have the Adobe AIR plug-in for Aptana Studio installed open the Plug-Ins Manager in Aptana Studio, locate and install it.
