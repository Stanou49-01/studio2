About Jaxer Views [edit]

This page gives an overview of the views that Aptana Studio provides for Jaxer development.

Introduction

Aptana Studio provides several views to help you with your Jaxer development work. These views help you manage your servers, including your Jaxer servers. The list below gives a summary of what these views are and what you can use them for:

  • Servers View - Use the Servers View to start, stop, manage, and monitor your Jaxer and other servers.
  • TailView View - Use the TailView View to view the logs of any process that generates log files.
  • Jaxer Activity Monitor View - Use the Jaxer Activity Monitor View to monitor the traffic to your internal Jaxer server.
  • Jaxer Shell View - The Jaxer Shell View provides you with an Ajax command-line console to interact with the Jaxer server.
