About Aptana projects [edit]

A project is a container used by the Workbench to group associated folders and files. Projects provide an organizational tool for your development work.

Aptana projects have a web focus and are typically groups of .js, HTML, and CSS files. Each project typically makes up a single web site, a part of a web site, or a JavaScript library.

Each project has its metadata stored in a project file called .project, stored in the project root directory.
