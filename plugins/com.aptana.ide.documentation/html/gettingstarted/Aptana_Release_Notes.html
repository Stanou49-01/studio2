Aptana Studio 1.0.1 patch (27-November-2007)

Build 004323 is a patch release for Aptana Studio 1.0, which includes the following major bug fixes:

  • CSS files being edited remotely not being saved (ASAP Issue STU-62)
  • Error when selecting a message in the message center (ASAP Issue STU-86)
  • Aptana Studio opens Aptana.exe in a tab when run from the desktop (ASAP Issue STU-188)
  • Aptana cannot find Firefox, and browse does not work for run configurations (ASAP Issue STU-353)
