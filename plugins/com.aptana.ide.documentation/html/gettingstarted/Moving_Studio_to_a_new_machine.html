Moving Studio to a new machine [edit]

Moving Aptana Studio to a new machine is reasonably simple as long as you have access to the previous installation.

  1. Back up your preferences and sync settings
  2. Back up your colorizations (only if you changed the defaults
  3. Install a fresh copy of Studio where the new instance of Studio should be
  4. Copy over the backup files
  5. Copy over your projects, as appropriate
  6. Launch the new Aptana Studio and install any plugins you use
  7. Import the backed up preference and settings
  8. Restart Studio
