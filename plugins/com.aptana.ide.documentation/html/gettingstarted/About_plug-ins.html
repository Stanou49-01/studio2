About plug-ins [edit]

A plugin is a component that you can install into Aptana Studio (Installing a plug-in) or the Eclipse IDE that extends the functionality of the base application. For example, if you install Aptana RadRails into Aptana, you will be able to use the Ruby Editor plus many other Ruby and Ruby on Rails tools within Aptana. Plugins allow you to keep all of your development tools in one environment rather than spread around separate applications for each programming function or task. Understanding how Aptana works with the Eclipse IDE has further details on the relationship between Studio and Eclipse.
