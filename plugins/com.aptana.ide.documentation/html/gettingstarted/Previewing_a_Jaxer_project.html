Previewing a Jaxer project [edit]

This Help topic describes how to preview a project that runs on the Jaxer server.

Introduction

The Built-in Preview Server included in Aptana also includes support for a Jaxer server. You will not need to configure any additional servers to preview a project that runs on a Jaxer server.

Instructions

Previewing a Jaxer project in the HTML editor

To preview a Jaxer project in the HTML editor:

  1. Open the main HTML file for your project in the Aptana HTML Editor.
  2. Click one of the browser preview tabs at the bottom of the HTML Editor window.

Aptana Studio displays an HTML preview of your Jaxer project.

Previewing a Jaxer project in a separate browser window

As long as you are using the built-in web server as part of your run configuration, you should be able to preview your project in a separate browser window without adding or configuring any additional servers.

You can double-check your run configuration by clicking the Run... button and selecting the browser that you want to use to run the project. You should have the Use built-in Web server option selected.

To preview a Jaxer project in a separate browser window:

  1. Open the main HTML file for your project in the Aptana HTML Editor.
  2. Click the Run.. button and select the browser that you want to use.

The browser preview displays the HTML page for your project.
