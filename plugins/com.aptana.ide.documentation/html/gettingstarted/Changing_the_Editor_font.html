Changing the Editor font [edit]

This page describes how to change the font in the Aptana Editors.

Introduction

The default font used in the Aptana Editors is a 10 point courier font. If you would like to change the size or font family for text that you type into the Editor, you can change the font in your preferences.

Instructions

To change the Editor font:

  1. From the Window menu, select Preferences... to open the Preferences window.
  2. On the Preferences tree on the left, expand General > Appearance and select Colors and fonts to change your fonts:

  3. On the list of fonts, expand the Basic folder.
  4. To change the Editor text font, select Text Font.
  5. Click the Change... button.
  6. Make your changes to the font, and click OK.
  7. On the Preferences window, click OK to save your changes.
