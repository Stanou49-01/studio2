What

Aptana Studio
The default panel for My Aptana gives you quick access to:
  • recently used projects and files
  • learning material and other resources
  • Pro license status

How

x

Open a Local Project

x

To open one of your local projects, find the name of the project you wish to open in the drop down menu, then click the open button.

Getting to My Aptana - Studio

x

  • If My Aptana is not open in Studio, click the home icon or your user name link at the right end of the Studio status bar
  • If you navigated to another panel in My Aptana, click the Studio icon
Use the other icons to access information about the Aptana Cloud, Jaxer, and Plugins.
