About the Aptana HTML Editor [edit]

Use Aptana's HTML Editor to code your HTML pages.

Key Features

  • Colorization provides visual cues.
  • Code Assist tag completion helps keep your HTML valid.
  • Preview tabs allows you to quickly preview pages in Web Browsers inside Studio.
  • HTML, JavaScript and CSS toolbars for quick tag insertions.


Link with Cursor

Enabling this feature (by clicking the double-sided arrow at the top right of the editor window) makes the editor toolbar switch between HTML, CSS and JavaScript tag sets based on the type of code on the row in which the cursor sits.
