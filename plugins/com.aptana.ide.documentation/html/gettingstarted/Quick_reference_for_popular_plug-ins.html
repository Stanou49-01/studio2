Quick reference for popular plug-ins [edit]

This page lists some of the most popular plug-ins and their update URLs for the Aptana community.

Reference

Name Description Update URL
Aptana Plug-in version of Aptana http://update.aptana.com/install/studio/3.2/
Aptana Monkey (Eclipse Monkey) Scripting tool for snippets and actions http://download.eclipse.org/technology/dash/update/
RadRails Plug-in version of RadRails/Aptana Rails http://update.aptana.com/install/rails/3.2/
PDT PHP Development Tools plug-in http://download.eclipse.org/tools/pdt/updates/
Flex Builder 2 (Adobe) Adobe Flex development tools for building Flash applications See http://www.adobe.com/go/flex_trial
Perforce Perforce source control plug-in for Aptana and Eclipse http://www.perforce.com/downloads/http/p4-wsad/install/
Subclipse Subversion source control plug-in for Aptana and Eclipse http://subclipse.tigris.org/update_1.2.x
