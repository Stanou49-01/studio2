What

The My Cloud screen lists your local projects which are deployed on Aptana Cloud.

My Cloud
Aptana Cloud is a scalable Elastic Application Cloud™ featuring fully stacked and integrated PHP app engines, Ajax/Jaxer app engines—and soon Ruby on Rails app engines—ready to use and ready to scale for your apps and sites as you need it.

Aptana Cloud plugs right into Aptana Studio to provide instant deployment, smart synchronization, and seamless migration as you scale with those same app engines running locally on your desktop for development and testing before you deploy. Aptana Cloud's team management, integrated source control, application monitoring and other turnkey services make working with the Cloud a breeze.


Deploying a Project to Aptana Cloud

x

Choosing a project and clicking "Deploy to Cloud" launches the deployment wizard. Next to each project a "Synchronize" button will launch the Smart Sync tool.


x

Your Cloud Sites

This list shows the name and current status for each of your sites, with the name a hyperlink that opens the site dashboard when clicked.

Statuses are:

  • Running: site is in a good state and your application should be functioning properly
  • Provisioning: site is in the process of being created and prepared for use, or you've resized and the update is being applied
  • Warning: a potential problem has been detected, check the site dashboard or Message Center for specifics
  • Error: a real problem exists on this site, check the site dashboard or Message Center for specifics

Getting to My Aptana - Cloud

x

  • If My Aptana is not open in Studio, click the home icon or your user name link at the right end of the Studio status bar and click the Cloud icon
  • If you navigated to another panel in My Aptana, click the Cloud icon
Use the other icons to access information about the Studio, Jaxer, and Plugins.
