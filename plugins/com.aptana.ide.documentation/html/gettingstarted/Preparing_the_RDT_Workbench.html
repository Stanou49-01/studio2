Preparing the RDT Workbench [edit]

This page explains how to set up your workbench for RDT.

Introduction

These instructions will explain how to verify that your Eclipse workbench is set up correctly for Ruby development.

Instructions

Verify the following before proceeding to the Important Ruby Settings Help topic:

  • You are starting with a new workbench installation with default settings.
  • You are familiar with the basic workbench mechanisms, such as views and perspectives.
