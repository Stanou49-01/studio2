About Local History [edit]

This page gives an overview of the Local History feature found in Aptana (and Eclipse).

Introduction

Use the Local History feature to track revisions of a file in your Aptana projects. Local History is a powerful feature that can help you troubleshoot your projects and even recover deleted files. Read through the following topics to learn more about Local History:

  • Setting Local History preferences
  • Comparing different versions of a file using Local History
  • Recovering a deleted file with Local History

You must have the Project View open to access Local History.
