Installing the iPhone plug-in [edit]

This page gives instructions for how to install the iPhone plug-in for the Aptana IDE.

Introduction

You will need to install the iPhone plug-in before you can access the iPhone development features.

Instructions

You can either install the plug-in for iPhone development into Aptana using the Aptana Start Page, the Plugins Manager View, or manually through creating a remote update site.

  • To install from the Aptana Start Page, simply click the link on the Start page, and follow the instructions in the wizard.
  • To install from the Plugins Manager View, select the iPhone plug-in, click the Install button Image:IconInstall, and follow the instructions in the wizard.
  • To install via a remote site, follow the instructions in Installing a plug-in, and use http://update.aptana.com/install/iphone/3.2/ as the URL for the remote update site.
