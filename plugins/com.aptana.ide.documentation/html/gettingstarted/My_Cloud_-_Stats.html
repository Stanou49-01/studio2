My Cloud - Stats [edit]

Aptana Cloud Docs Home

What

x

Cloud stats
The cloud stats screen has graphs for usage of
  • CPU
  • Memory
  • Swap
  • Disk

as well as graphs for web stats including

  • Successful pages
  • Successful hits
  • Pages with referrers
  • Bytes transferred
  • Total errors
  • Total visits
  • Total unique visitors

The Advanced Web Stats button opens a special web page that provides access to extremely detailed access logs reports based on the AWStats/JAWStats packages.

How

  • Use the radio buttons on the right side to switch between the available graphs.
  • Use the radio buttons on the bottom of the screen to change the chart time span.
