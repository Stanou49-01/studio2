Creating a new iPhone project [edit]

You can create an iPhone project using the project wizard, just like you would for any other project in Aptana.

If you have not already done so, you will need to install the iPhone plugin before you can create an iPhone project.

To create an iPhone project:

  1. From the File menu, select New > Project... to open a New Project wizard.
  2. On the first screen of the New Project wizard, expand iPhone and select iPhone Project, and click Next to go to the New iPhone Project screen.
  3. In the Project name field, type a name for your new project and click Finish.

The Aptana project wizard creates a new iPhone project for you. Your new project now appears in your Project View. You can preview your project by clicking the iPhone Preview tab at the bottom of the editor window.
