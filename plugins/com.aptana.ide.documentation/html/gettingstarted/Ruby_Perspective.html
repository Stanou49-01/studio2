Ruby Perspective [edit]

This page gives an overview of the Ruby Perspective that is part of Ruby development tools.

Introduction

The Ruby Perspective is designed for working with Ruby projects. It consists of an editor area and the following views:

  • Ruby Resources View
  • Ruby Outline View
  • Ruby Console View
  • Task List
  • Problems View
