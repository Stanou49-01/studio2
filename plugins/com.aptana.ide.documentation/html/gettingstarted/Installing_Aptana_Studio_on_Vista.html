Installing Aptana Studio on Vista [edit]

This page describes how to install the standalone Aptana Studio on Windows Vista.

Introduction

If you are installing the standalone Aptana Studio on a computer running Windows Vista, you will want to set Aptana to run as Administrator if you wish to update or install new plugins.

Instructions

Running Aptana Studio(M8 or later) on Windows Vista

To run Aptana on Windows Vista after installation:

  1. Right-click the Aptana shortcut, and select Properties from the context menu.
  2. Go to the Shortcut tab.
  3. Click the Advanced button
  4. Check the Run this program as an administrator box.
