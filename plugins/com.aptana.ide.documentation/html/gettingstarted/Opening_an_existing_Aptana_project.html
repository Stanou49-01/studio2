Opening an existing Aptana project [edit]

This page describes how to access an existing Aptana project.

Introduction

When you start Aptana, all of the projects in your workspace will automatically open, so you should not have to "open" a project as a separate step. All of your projects should be visible in the Project View.

Instructions

Use the following tips to access an existing Aptana project if you do not immediately see it in your Project View.

  • Check the project tree in the Project View. If you do not see the Project View on your workbench, see Displaying the Project View.
  • If you are currently in a different workspace than the one where the project exists, switch your workspace to the one that the project was created in. See Switching your workspace to learn how to switch your workspace.
