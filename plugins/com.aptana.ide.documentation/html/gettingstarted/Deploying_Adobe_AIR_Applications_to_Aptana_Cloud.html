Deploying Adobe AIR Applications to Aptana Cloud [edit]

Prerequisite: Create an Adobe Air Application

  1. In the Project view, right click on your application and choose Deploy to Cloud (complete the Cloud Deployment wizard to setup your domain and service)
  2. Package and publish your AIR application
  3. Create an application badge--be sure to update your Badge File Name to index.html to make it the app's default page
  4. Check that your application URL specifices the domain name you use on your Aptana Cloud site
  5. In the Project view, right click on your application and sync your project to the Cloud
  6. Point a web browser to your domain and check your application
