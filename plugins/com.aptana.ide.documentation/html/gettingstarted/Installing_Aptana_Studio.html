Since Aptana Studio is based on the Eclipse platform, you can install Aptana Studio as a standalone application (recommended) or, if you're already an Eclipse user, you can add Studio as a plugin into your Eclipse v3.2 or higher installation.

  • Standalone Aptana Studio users: Complete the instructions on this page
  • Plugging Aptana Studio into an existing Eclipse configuration

If your work will primarily be web development-related, we recommend installing the standalone Aptana Studio. If your work involves Java development or non-web-related work, we recommend installing Aptana as an Eclipse plugin.

Prerequisite: Ensure your machine meets the Aptana System Requirements.

Pro Users: After completing the base installation, see Upgrade to Aptana Studio Pro for instructions on adding your Pro license.

To download and install Aptana Studio:

  1. Download the installer or zip file for your operating system
  2. Launch the installer
  3. When Studio runs you'll get the chance to install Aptana's web development plugins

Windows Vista Install Note

On Microsoft Windows Vista, Aptana Studio must run as Administrator to have privileges to install or update new plugins.

  1. Right-click the Aptana Studio shortcut, and select Properties from the context menu.
  2. Go to the Shortcut tab.
  3. Click the Advanced button
  4. Check the Run this program as an administrator box.

Customize Your Installation

The first time you launch Studio after installation you'll be shown the Install Additional Features wizard, which lets you easily add programming language (PHP, Jaxer, Ruby on Rails, Python), Ajax library (including jQuery, Dojo, YUI, Mootools and EXT JS) and source control (Subversion, Git) plugins. You can check the Do not show again checkbox so the wizard is not run on subsequent launches and if you do that you can bring it as needed by choosing Install Additional Features from the Aptana menu.

Default Locations

Aptana Studio uses these default locations for installation, configuration files and workspace.

Mac OS X

  • Installation: /Applications/Aptana Studio 1.3/
  • Workspace: ${HOME}/Documents/Aptana Studio Workspace
  • Configuration Area: ${HOME}/Library/Application Support/Aptana/Aptana Studio 1.3/configuration

Windows XP

  • Installation: c:\Program Files\Aptana Studio 1.3
  • Workspace: @user.home\My Documents\Aptana Studio Workspace
  • Configuration Area: @user.home\Application Data\Aptana\Aptana Studio 1.3\configuration

Windows Vista

  • Installation: c:\Aptana Studio 1.3
  • Workspace: @user.home\Documents\Aptana Studio Workspace
  • Configuration Area: @user.home\AppData\Roaming\Aptana\Aptana Studio 1.3\configuration

Linux

  • Installation: Where you unzip the downloaded file, typically ${HOME}/Aptana Studio 1.3
  • Workspace: ${HOME}/Aptana Studio Workspace
  • Configuration Area: ${HOME}/.Aptana/Aptana Studio 1.3
