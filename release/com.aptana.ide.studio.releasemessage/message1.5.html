Aptana Studio Andretti Alpha #3 Released!

Welcome to Aptana Studio Andretti Alpha

Welcome and thank you for testing the Alpha build of the next Aptana Studio release, code-named Andretti.

Aptana Studio "Andretti" is built on Eclipse 3.4.
Unlike previous Studio versions, Andretti is no longer compatible with Eclipse 3.2 and 3.3. If you're running a version of Studio prior to Andretti, you will not be able to update to Andretti; instead, you will need to reinstall.

How you can help
We want the final Andretti release to be stable and high quality. You can help us achieve that goal by testing new functionality, looking out for regressions, filing bugs and enhancement requests, and giving us your feedback in the bugbase and the forums. You can now log issues and enhancement requests from inside Studio by choosing Help > Aptana Help > Troubleshooting > Submit Aptana Bug or Feedback....

Release Notes

What's New in Alpha #3

  • Updated XULrunner 1.8 to XULRunner 1.9
  • Updated Windows bundled Multilingual JRE to 1.6.0_13
  • SFTP - Public Key Authentication
  • More Database Fixes
  • More FTP and Synching Fixes

What's New for Andretti Releases

  • Eclipse 3.4 is now the base platform for Aptana Studio.
  • Install only the plugins you need with the Install Additional Features wizard, which runs on launch and from the Help menu. Options include, but are not limited to:
    • Ajax library support, including jQuery, Dojo, Ext JS, YUI, and many others
    • Aptana Jaxer v1.0.3 (Release Notes)
    • PHP
    • PyDev
    • Git
  • Browse and manage your databases with the new Database Explorer.
  • File bugs and enhancement requests from within Studio by going to Help > Aptana Help > Troubleshooting > Submit Aptana Bug or Feedback...
  • The web server configuration now includes the DocumentRoot property, simplifying the use of some web frameworks.
  • More FTP and Synchronization Fixes
  • Fixed Aptana Studio Restart Issues
  • Fixed Managed Sites bug that caused Plugin Install Failure
  • Full Path now in the Aptana Studio Title Bar
  • Adobe Air Plugin now works

Known Issues

  • This is an Alpha build! Its stability is not guaranteed, and its feature set is not complete. Save often, and avoid using this build for critical work.
  • We do not recommend using the Database Explorer with Oracle databases at this time.
